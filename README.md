SH-SETUP-SCRIPTS
================

Some scripts I wrote for system administration tasks.

## Help

linux-h-bindhosts

    Usage: linux-h-bindhosts -u | NAMES....
    
    Temporarily map names to 127.0.0.1 in "/etc/hosts".

runit-h

    Usage: runit-h [-v][-D DESTDIR] ...
    
      -a NAME PROGRAM ARGS... : Create a normal waiting service.
    
    Create and manage services using runit.

systemd-h

    Usage: systemd-h [-v][-D DESTDIR][-d DESC] ...
    
      -a NAME PROGRAM ARGS... : Create a normal waiting service.
    
    Create and manage services using runit.
